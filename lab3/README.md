# Лабораторна робота №3: Візуалізація даних

## Опис завдання
- Вибір унікального датасету (Automobile Data Set з репозиторію UCI).
- Очищення даних (Data Cleaning) від пропущених значень.
- Побудова графіка залежності одного атрибута від іншого.
- Побудова гістограми, що показує кількість елементів у 5 заданих діапазонах.
- Візуалізація багатовимірних даних (теплова карта, pairplot, boxplot).
- Загальна кількість побудованих графіків становить 5-8.

## Вимоги до системи
- **ОС:** Windows 10/11
- **Інтерпретатор:** Python 3.9+

## Інструкція із запуску

1. **Створення віртуального середовища:**
   - Windows/Linux/macOS:
     `python -m venv venv`

2. **Активація віртуального середовища:**
   - Windows (PowerShell):
     `.\venv\Scripts\activate`
   - Linux / macOS:
     `source venv/bin/activate`

3. **Встановлення залежностей:**
   `pip install -r requirements.txt`

4. **Запуск:**
   `jupyter notebook`
