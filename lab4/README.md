# Лабораторна робота №4: Інтерактивна візуалізація гармоніки

## Опис проекту
Ця програма побудована на базі бібліотеки `matplotlib` та призначена для візуального аналізу гармонічних сигналів. Вона дозволяє в реальному часі змінювати параметри сигналу, накладати шум та перевіряти ефективність фільтрації.

## Функціонал
- **Редагування гармоніки:** зміна амплітуди, частоти та фази.
- **Робота з шумом:** налаштування середнього значення та дисперсії шуму.
- **Фільтрація:** застосування фільтра Баттерворта (Low-pass) для очищення сигналу.
- **Інтерактивність:** повзунки (sliders), чекбокси для керування видимістю та кнопка скидання параметрів (Reset).
