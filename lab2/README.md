# Лабораторна робота №2

## Опис завдання
Робота з Pandas, обробка даних NOAA та датасету споживання електроенергії.

## Вимоги до системи
- **ОС:** Windows 10/11
- **Інтерпретатор:** Python 3.9+

## Інструкція із запуску
1. Активуйте віртуальне середовище у кореневій папці проєкту:
   `.\venv\Scripts\activate`
2. Перейдіть у папку з лабораторною та встановіть залежності:
   `pip install -r requirements.txt`
3. Запустіть середовище для перегляду файлів `.ipynb`:
   `jupyter notebook`
